# Rules and Constraints

1. The system must use semantic embeddings to retrieve relevant content.
2. Retrieval must prioritize semantic relevance rather than keyword matching.
3. Ranking should reflect true similarity between query and document meaning.
4. All retrieval steps must remain deterministic.

Participants may modify:

- Chunking strategy
- Similarity computation
- Ranking logic
- Retrieval selection

Important:

The program should continue to run successfully after improvements.
The goal is improving algorithm correctness, not fixing runtime errors.
