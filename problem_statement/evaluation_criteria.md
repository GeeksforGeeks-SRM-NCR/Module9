# Evaluation Criteria

Submissions will be evaluated based on:

1. Retrieval Accuracy:
   - Relevant documents appear in top results.
   - Irrelevant documents are minimized.

2. Algorithm Understanding:
   - Correct similarity metric usage.
   - Proper ranking logic.

3. Efficiency:
   - Avoid unnecessary recomputation.

4. Code Quality:
   - Clear reasoning behind improvements.
